# RosterReady - Softball Tryout Management System

A comprehensive, professional web application for managing softball tryouts with role-based dashboards for players, parents, coaches, and administrators.

## 🚀 Live Demo

Visit the live application: [Your GitHub Pages URL]

## ✨ Features

### For Players
- View tryout schedule and station assignments
- Check-in functionality
- Real-time scores and performance feedback
- Personal profile management
- Coach notes and improvement areas

### For Parents
- Monitor child's tryout progress
- View scores and evaluations
- Access tryout schedules and locations
- Communication with coaches

### For Coaches
- Station-based scoring interface
- Player evaluation tools (1-10 scale)
- Quick notes and comments
- Assigned station management

### For Administrators
- **Organization Management** - Create and manage multiple organizations
- **Tryout Setup** - Configure player count (10-200), duration, age levels (8U-18U)
- **Station Management** - Create and assign evaluation stations
- **Staff Management** - Add coaches and evaluators
- **Player Roster** - Import via CSV or add manually
- **Auto-Assignment** - Automatically distribute players to stations
- **People Directory** - Searchable directory of all members (admins, coaches, players, parents)
- **Group Messaging** - Send updates to specific groups or all members
- **Invite System** - Share links to invite new members

## 🎨 Design Features

- **Professional Georgia Font** - Elegant serif typography throughout
- **Vibrant Color Palette** - Red, blue, purple, teal gradients
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Interactive Elements** - Sliders, buttons, search, and filters
- **Clean UI** - No emojis, professional SVG icons
- **Role-Based Views** - Customized dashboards for each user type

## 📦 Installation

### Option 1: GitHub Pages (Recommended)

1. Fork this repository
2. Go to Settings → Pages
3. Select "main" branch as source
4. Your site will be live at `https://yourusername.github.io/rosterready/`

### Option 2: Local Development

1. Clone the repository:
```bash
git clone https://github.com/yourusername/rosterready.git
```

2. Open `index.html` in your web browser

That's it! No build process, no dependencies, no installation required.

## 🎮 Quick Start

1. Open the application
2. Click one of the quick login buttons:
   - **Player** - See player dashboard
   - **Parent** - See parent/guardian view
   - **Coach** - See coach evaluation tools
   - **Admin** - Access full management system

3. Explore features:
   - Navigate through tabs
   - Use the slider to set player count (10-200)
   - Search the people directory
   - Send group messages
